Example File Format

JSON format:

    [
        {
            "id": "123",
            "name": "Example Product 1",
            "description": "Product description",
            "image_url": "path/to/image1.png",
            "price": "9.99",
            "category_id": 1,
            "featured": 1
        },
        {
            "id": "124",
            "name": "Example Product 2",
            "description": "Product description",
            "image_url": "path/to/image2.png",
            "price": "19.99",
            "category_id": 2,
            "featured": 0
        }
    ]

CSV format:

    id, name, description, image_url, price, category_id, featured
    123,"Example Product 1","Product description","path/to/image1.png",9.99,1,1
    124,"Example Product 2","Product description","path/to/image2.png",19.99,2,0

Text format (tab-separated):

    id      name                description             image_url               price       category      featured
    123	    Example Product 1	Product Description     path/to/image1.png      9.99        1             1
    124	    Example Product 2	Product Description     path/to/image2.png      19.99       2             0
