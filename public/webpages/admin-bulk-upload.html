Example File Format

JSON format:

    [
        {
            "productId": "123",
            "productName": "Example Product 1",
            "description": "Product description",
            "category": "Electronics",
            "imageUrl": "path/to/image1.png",
            "price": "9.99"
        },
        {
            "productId": "124",
            "productName": "Example Product 2",
            "description": "Product description",
            "category": "Books",
            "imageUrl": "path/to/image2.png",
            "price": "19.99"
        }
    ]

CSV format:

    productId, productName, description, category, imageUrl, price
    123, Example Product 1, Product description, Electronics, path/to/image1.png, 9.99
    124, Example Product 2, Product description, Books, path/to/image2.png, 19.99

Text format (tab-separated):

    productId	productName	        description             Category        imageUrl                Price
    123	        Example Product 1	Product Description     Electronics     path/to/image1.png      9.99
    124	        Example Product 2	Product Description     Books	        path/to/image2.png      19.99
